Usage

If you have Substance Designer open and are in the Graph View, you just need to press the left mouse button while holding a hotkey. Alternatively, you can also hold left Click and press the hotkey button. Many hotkeys have multiple uses, depending on whether or not modifying keys like shift are pressed.
If you want a complete list of hotkeys, you can look into the script itself. In there, you will also find more detailed explanations.

Features

Create Nodes without typing their names!

- Hold a hotkey and click anywhere to create a new Node.
- Since Shift, Alt and Ctrl can be used, there is room for more hotkeys than you'll ever need.
- Drag the pin of a Node and press a hotkey, to create a new Node. An automatic connection will be made.
- Hold a hotkey and click on an existing connection. A Node will be connected in between.
- A new hotkey can only be activated after the old one has been released - you will never accidentally create a Node twice.
- The Script is only active if Substance Designer is the active Window.
- Adding new hotkeys is very easy - even if you are not a programmer.
- Common Textures can be created by holding one key and typing a number (e.g.: c + 1 for Clouds 1).

How it works - How to customize

The core of the functionality is a single AutoHotkey Script. AutoHotkey is a great scripting language for automating simple tasks in any programm. I used it to create Hotkeys which emulate more complex input from the keyboard.
e.g.: h + LeftMouseButton -> Spacebar, 'Histogram Scan', Enter

To make creating more hotkeys easy and comfortable I created a function which takes a string as a parameter. The string contains multiple parameters which are separated by commas. First comes the hotkey, followed by which Substance Designer Node should be created. After that come optional parameters which define what should happen, if the shift/alt/control keys are pressed.
e.g.: NodeFunction("b, Blend, Blur HQ")
-> If just b is pressed, a Blend node will be created.
-> If shift and b are pressed at the same time, a Blur HQ Greyscale will be created. -> If shift and b are pressed while a pin from a color node is dragged, a Blur HQ Color will be created.

To create your own shortcut, you simply have to open the script, navigate to the function in which all the calls to the NodeFunction are and create an additional call. One simple line of code is all you need.
