Custom Raymarcher

Overview

A Raymarcher built completely from scratch inside UE4.

Mandelbulb Fractal:

Raymarching Demo:



Features:
- Occlusion culling with default meshes
- Add custom objects simply by placing them in the scene (no additional code required)
- Write any UE4 HLSL code inside editor and see the result in the scene almost immediately
- Works with any number of objects
- Various modes similar to realtime boolean operations (Additive, Subtractive, Union)

- Materials are created seperately and can be assigned to individual objects in the editor
- Sphere- and triplanar-mapping

- Everything is inside a plugin. Additing it to a new project is very simple.
