This Website

Why list the website as a project?

Because quite a bit of programming went into it. And because I taught myself HTML, CSS and Javascript to create it.
The only resource I used was This Template from W3Schools. You may notice that visually, it looks almost the same as my website. Except for the sidebar however, it got no functionality.

All these functions were made by me for this website:

- Filtering projects by group/solo
- Resizing project panel's height
- Stopping/Starting videos on click
- Displaying a play button on videos before being played
- Loading project content dynamically (at the bottom of the page and in the sidebar)

Even though I used CSS classes from W3Schools, I also wrote around 17 of my own.
HTML was relatively intuitive and easy for me, right from the start. It being only one of three languages I had to learn, made it a challenge nonetheless.

All in all, I am really proud of this website and hope it might help convince someone (you?) to give me a job in the Game Industry!
